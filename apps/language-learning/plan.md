# Language Learning App


I hate all of the public application for language learning (Duolingo, etc). I want to make my own light weight application for my own style of learning. This application is mostly focused on teach through reading and writing as opposed to speaking. I learn best by reading and writing, so once I feel good about that, I will work on speaking and listening.

I am a native English speaker and am interested in learning Spanish and Farsi because my Dad speaks Spanish and my Mom speaks Farsi. All UX created should natively support both RL and LR languages.


Core things the app should do:
- Progressively teach and keep track of my understanding of my vocabulary in the language.
    - There should be a database of all the words in the language categorized by importance and commonality in the language.
    - I should have a table of words that I have learned scored on a progressive scale for my ability to use and understand the word.
- There should be a flash card system that helps me learn the words with pictures and common sayings. As I learn with the flash cards, it should use a progressive memory scoring system like in Anki to help with repetition of the words in flash cards.
- I want a feature where I can upload the lyrics of a song in the language and then the application makes me work through translating that song into a language that I already know. 
- I want an LLM based feature that generates stories for me to read in the language using my scored vocabulary and general life interests.
- I want to upload transcripts of videos that I have watched in that language that then get added to my stored vocabulary.
- For grammar, we will need a defined path for the structures that I will need to learn. The system should attempt to progressively teach me grammar rules based on my progress through the language.
